# Drumstalotājs

Topdown turn-based tactical shell shooter-launcher.

## Description

Drumstalotājs (*The Crumbler* in Latvian, for player is the one) is a minimalistic wargame with a focus on the artillery and mortar strikes. The player's aim is to retake in-game's Rocky Island and player commands an unit from mortar teams to artillery batteries in each level on the island with aim to destroy targets to support the player's army's efforts to retake the island.

Project made as a diploma work for Liepāja State Technical University
